# Edge-AI Context-Aware Irrigation Simulator

A standalone edge-device simulation designed to optimize crop irrigation in rural farming environments without relying on cloud or internet connectivity.

## 🚀 How to Run Locally
1. Install dependencies: `pip install streamlit pandas`
2. Run the app: `streamlit run app.py`

## 🧠 How it Works
The system uses a lightweight, multi-variable heuristic scoring algorithm ideal for microcontrollers (like ESP32/Arduino)
to make precise watering decisions based on soil moisture, temperature, humidity, and growth stages.
